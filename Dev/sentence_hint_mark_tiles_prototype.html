“Mark 2 correct tiles” — visual options

The PvE hint reveals 2 correct tiles and the slot each belongs in. Same demo sentence in every card. Hit “Fire hint ✨” on each to see the effect, then pick the treatment you like.

Demo prompt: “My sister reads a book.” → Mi · hermana · lee · un · libro (5 slots, 8-tile pool). The hint reveals slot 2 = “hermana” and slot 5 = “libro”. Amber = “revealed by hint” (kept distinct from the green you’d get from a correct Confirm).

Option 1 — Corner badge + glow minimal

Revealed tiles get an amber glow and a corner badge with the slot number. Reuses the existing order-badge pattern, just in a hint colour.

Good: tiniest change, familiar. Watch: “slot 2” is implicit — no on-screen slots to anchor the number to.

Option 2 — Slot tray link explicit

A numbered slot tray sits above the pool. Firing the hint lights up the two target slots and their matching tiles in the same colour/number — you literally see “this tile → that slot”.

My sister reads a book.
58 sec
tile ↔ slot both lit

Good: position is unmistakable. Watch: adds a permanent slot tray to the board (a layout change for all rounds).

Option 3 — Ghost in the slot strongest

The revealed word appears faintly inside its target slot, and the tile gets a green check. You see the answer forming in place.

My sister reads a book.
58 sec
word previewed in slot

Good: clearest of all. Watch: shows the word text twice; arguably gives away more than the others.

Option 4 — Floating “→ slot N” tag compact

No tray. Each revealed tile wears a little pill that reads “→ #2”. Compact and self-explanatory without changing the board layout.

My sister reads a book.

tag says the slot

Good: no layout change, position is spelled out. Watch: pill can crowd a small tile on narrow phones.
